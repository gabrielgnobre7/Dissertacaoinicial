# Dissertacao Inicial
Compartilho um pouco dos meus avanços da minha dissertação com tratamento da base de dados de marcas e algumas análises estatística simples.
A saber, faço mestrado em economia, e meu tema de pesquisa é: inovações do setor de serviços. Conforme a literatura econômica, as marcas são os indicadores mais apropriados para medir a inovação no setor de serviços. Por isso, irei explorar esta base de dados disponiblizada pelo INPI (Instituto Nacional de Propriedade Intelectual) para entender os apectos econômicos e regionais que produzem mais inovações regionais.
